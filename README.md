# Gaming-Player-Engagement-Analysis
This repository contains an exploratory data analysis and insights derived from a comprehensive online gaming player behavior dataset. The analysis focuses on understanding player demographics, engagement patterns, gameplay habits, and factors influencing player retention.

# Dataset Overview

The dataset includes 40,034 player records with 13 features, covering:

Player demographics (age, gender, location)

Gameplay characteristics (game genre, difficulty, player level)

Engagement metrics (playtime, sessions per week, session duration)

Monetization behavior (in-game purchases)

A target variable representing player engagement level (High, Medium, Low)
